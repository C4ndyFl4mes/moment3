SCSS utvärdering

För- och nackdelar

Fördelarna till att använda SCSS istället för vanlig CSS är

  • Man kan använda Nesting som underlättar att organisera.
  • Mixins kan ta emot argument som kan minska repeterande kod.
  • Arv kan användas för att ärva egenskaper från andra selektorer.
  • Partials kan användas för att strukturera upp koden i olika filer.
  • Bra för stora projekt.

Nackdelarna är

  • Komplex för nybörjare.
  • Man måste använda en kompilator då webbläsaren kan inte direkt läsa SCSS kod.
  • Om man är slarvig med t.ex. @extend kan det orsaka onödigt stora filer.
  • Inte nödvändig för små projekt.

Hur det gick

Det var och är fortfarande ganska förvirrande att veta när en mixin eller inheritance skall användas. Jag har nog inte gjort det på ett bra sätt, men har försökt ganska mycket med att det ska bli så lite kod som möjligt. Mörkt/ljust tema var lite besvärtligt med att inte kunna göra extend över queries.
